# Statistical-Learning-Project
## Abstract
This report analyzes residential property values in Taipei, Taiwan, using the
Real Estate Valuation
dataset from the [UCI Machine Learning Repository](https://archive.ics.uci.edu/dataset/477/real+estate+valuation+data+set).

The study first explores the structure of the housing market through Principal Component Analysis (PCA) and clustering, revealing three economically meaningful groups of properties largely differentiated by accessibility to MRT stations, neighborhood amenities, and building age.

A supervised learning framework is then applied to assess the predictive power of structural and locational features on house prices. Among the models considered—linear regression, Neural Networks, and Random Forest—the latter provides the best performance, confirming the importance of non-linear spatial effects.

Overall, the results highlight that proximity to MRT stations and geographical location are the strongest determinants of housing prices, while structural attributes such as house age play a secondary role. These findings illustrate both the segmentation of Taipei’s housing market and the relevance of flexible models for price prediction.
